About Mythreya

This application has used Swiss Ephemeris package as per GNU GPL software license terms. All planetary positions are computed using The Moshier Ephemeris. This is a semi-analytical approximation of the JPL planetary and lunar ephemerides DE404, developed by Steve Moshier. Its deviation from JPL is below 1 arc second with the planets and a few arc seconds with the moon. This may be sufficient accuracy for most purposes, since the moon moves 1 arc second in 2 time seconds and the sun 2.5 arc seconds in one minute. All panchang elements are computed using the planetary positions as obtained above. Hindhu festival events are determined using rules as stated in Dharama Sindhu / Nirnaya Sindhu.
